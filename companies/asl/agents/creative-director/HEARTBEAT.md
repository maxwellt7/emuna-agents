# Creative Director — HEARTBEAT

## Cycle

This heartbeat runs on a recurring loop. Each cycle follows these steps in order.

### Step 1: Identity Check

- Call the Paperclip API to confirm identity and retrieve agent configuration.
- Verify role, company assignment, and active status.
- If identity check fails, halt and report the error.

### Step 2: Get Assigned Tasks

- Query the task board for all tasks assigned to Creative Director.
- Filter for open and in-progress tasks.
- Prioritize by due date, then by task priority flag.

### Step 3: Research the Brand and Offer

For each assigned creative task:

- Review the brand profile, offer details, and target avatar.
- Study the competitive landscape: pull recent ads from competitors in the same vertical.
- Identify what angles and hooks are currently running in the market.
- Review any existing performance data for this brand's creatives (CTR, CVR, CPA, ROAS).
- Note brand compliance requirements or restrictions.

### Step 4: Develop Angles

- Produce 3-5 distinct creative angles for the task.
- Each angle must include:
  - **Angle name**: A short label (e.g., "Clinical Proof," "Fear of Missing Out," "Transformation Story").
  - **Strategic rationale**: Why this angle should resonate with the target avatar.
  - **Hook options**: 2-3 specific hook lines or opening frames.
  - **Mechanism**: The believability element that supports the angle.

### Step 5: Produce Creative Briefs

For each approved or top-priority angle, produce a full creative brief:

- **Angle and hook** (exact language)
- **Headline and subhead variations** (3+ options)
- **Body copy direction** (key talking points, proof elements, emotional arc)
- **Visual direction** (thumbnail concept, color guidance, imagery style, video pacing notes if applicable)
- **CTA language** (specific button text or closing line)
- **Format specs** (ad type, dimensions, duration if video)
- **Compliance notes** (any claims that need disclaimers, restricted language)

### Step 6: Review Performance and Flag Fatigue

- Pull performance data on all active creatives.
- Flag any creative where CTR has dropped more than 20% from peak or frequency has exceeded thresholds.
- Recommend kill, iterate, or scale for each active creative.
- Ensure replacement angles are queued for any creative approaching fatigue.

### Step 7: Report and Comment

- Comment on each completed task with the deliverables (briefs, angle decks, performance reviews).
- Update task status to reflect completion or next action needed.
- Summarize cycle output: number of briefs produced, creatives reviewed, fatigue flags raised.
